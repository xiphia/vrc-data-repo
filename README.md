# Personal repository for VRChat

VRChat からの利用を想定したデータを配置するリポジトリ

## VirtualShikoku

VRChat 四国集会にて利用するデータ群を収めたディレクトリ

### `data.json`

四国パズル等から利用する、四国四県 95 市町村のデータをまとめた JSON ファイル
